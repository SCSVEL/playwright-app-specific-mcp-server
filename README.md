# playwright-app-specific-mcp-server
This is to design the application specific layers, and to incorporate them part of MCP tools to utilize from the AI Coding Agents or clients to speed up testing

## Prerequisites
- Python 3.11 or higher
- Playwright
- MCP Server

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python myapp_mcp_tools.py
```
